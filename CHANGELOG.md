# Changelog

## 0.1.1

- Added Sentinel-2 NDVI and NDWI derived-product presets.
- Added clearer band labels such as `Red (B4)` and `Green (B3)`.
- Added VRT wrapper output for better QGIS band naming behavior.
- Added optional single-band analysis layers for multiband downloads.
- Improved plugin metadata and packaging readiness for public release.

## 0.1.0

- Initial release with Earth Engine authentication.
- Added GeoTIFF download and automatic QGIS layer loading.
- Added dataset presets, region selection, and code preview.
